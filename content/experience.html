This is a small example of work I have done in React to create a live weather tracking app for the Toronto Area. This web application was built using the Open Weather API.

Using Graphql and MongoDB, this project was able to store user input data to create unique hotels, users and bookings. A user would be to enter their own data to organize different hotels and bookings for individual users. Hotels can be found by hotel name, hotel city or a general query which returns all hotels listed in the database.
