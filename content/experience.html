Open Weather API

This online weather app was built using React and the Open Weather API. It is a live updating application that shows the current weather in the Toronto Area. This was an individual project developed for Full Stack Development course.

GraphQL

This is a GraphQL assignment that uses MongoDB to store and retrieve all user inserted data to and from the webpage. This application allows users to create hotels with unique ID's to be stored in MongoDB for the purposes of creating a booking system. Hotels, bookings and users are created by users of this application to fulfill a specific schema.
