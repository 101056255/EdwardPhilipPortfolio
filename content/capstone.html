Stray No More

The Stray No More Mobile Android Application is a group effort for the Capstone course at George Brown College. It was designed and deployed with the work of myself and three of my fellow colleagues.

Here is an example of the login screen for the app, this work is a big jump from my previous experience in Android Studio when developing the Restaurant Guide. Using Firebase, this application is able to have user authentication and realtime database storage of all relevant data.

Stray No More

The registration page contains the unique user information inputted and saves it to a realtime database which allows the user's personal details to be linked to their profile. This is done by combining the unique UID created by Firebase Authentication to the database.

Stray No More

The map page in the application will show live up-to-date markers placed by other users to list last known locations of lost pets, stray animals or provide details and locations of animal shelters and animal control facilities.

The map is a live Google Maps fragment similar to that in the Restaurant Guide.
