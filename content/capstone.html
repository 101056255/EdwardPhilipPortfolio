The Restaurant Guide android application is a simple introduction into using Android Studio. The basis of the app is to allow users to create a dynamic list of restaurants which can be added and removed by the user.

Stray No More

Once a user has tapped one of the restaurants on the list a page will be opened with more detail about the restaurant. This includes the restaurant name, address, phone number, description and tags which are all entered in by the user. A Google Maps view is also included on the page using the address inserted by the user to mark the location of the restaurant allowing the user to seamlessly find the directions

From this screen the user can edit any details they would like, as well as share the details of the restaurant on social media platforms such as Facebook, Twitter or by email.
